# Админка каталога

Редактирование данных как в `data/catalog.json` в браузере.

## Запуск через локальный сервер (рекомендуется)

Из **корня репозитория**:

```bash
python3 -m http.server 8765
```

Откройте: `http://127.0.0.1:8765/_admin/` — каталог подгружается из `data/catalog.json` при открытии и по кнопке **«Обновить из data»**.

## Запуск без сервера (file://)

В режиме `file://` автозагрузка отключена. Используйте кнопку **«Загрузить JSON…»** и выберите `data/catalog.json`.

## Сохранение

После правок нажмите **«Скачать catalog.json»** и замените **`data/catalog.json`** в проекте.

## Почему не попадает на GitHub Pages

- Папка **`_admin`**: у Jekyll каталоги с `_` в корне не публикуются как часть сайта.
- В корне **`_config.yml`** с `exclude: [_admin, tools]`.

Если в корень добавить **`.nojekyll`**, Jekyll отключается, `exclude` не действует, и **`_admin` может стать доступен по URL**. Тогда нужен отдельный деплой без этой папки.
